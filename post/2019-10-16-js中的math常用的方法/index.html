Js中Math()的使用

Posted by     "Skwings" on Tuesday, October 15, 2019

TOC

    • 一、Math
      • Math中常用的属性和方法
        • 1.Math.abs([value])
        • 2.Math.ceil/floor([number value])
        • 3. Math.round()
        • 4. Math.max/min([val1],[val2]…)

一、Math

数学函数：但是他不是一个函数，他是一个对象，对象中存储了许多的属性方法，所以被称为数学函数。

typeof math //=> object; Math是一个对象

Math = {
    ... //Math的方法属性。
}

Math中常用的属性和方法

1.Math.abs([value])

Math.abs()的作用是获取接受参数的绝对值（正数或者0）

console.log(Math.abs(-12.5)); //=>12.5
console.log(Math.abs(-1px)); //=>NaN
console.log(Math.abs(true)); //=>1

2.Math.ceil/floor([number value])

把一个数向上取整/向下取整

console.log(Math.ceil(12));//=>12
console.log(Math.ceil(12.1));//=>13
console.log(Math.ceil(12.9));//=>13
console.log(Math.ceil(-12.5));//=>12

console.log(Math.floor(12));//=>12
console.log(Math.floor(12.1));//=>12
console.log(Math.ceil(12.9));//=>12
console.log(Math.ceil(-12.5));//=>-13

3. Math.round()

四舍五入

console.log(Math.round(12));//=>12
console.log(Math.round(12.1));//=>12
console.log(Math.round(12.9));//=>13
console.log(Math.round(-12.5));//=>12  负数中.5舍去

4. Math.max/min([val1],[val2]…)

获取最大/最小值

console.log(Math.max(12,5,7,2,8,2,4,5));  //=>12 取最大值
console.log(Math.min(12,5,7,2,8,2,4,5));  //=>2 取最小值
